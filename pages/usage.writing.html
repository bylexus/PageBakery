Writing Pages

A "Page" in The Page Bakery is generated with two components:

  • The Layout, which is the surrounding "Skeleton" of a page. See Defining Layouts for more details on layouts.
  • The page content, which is the, well, content, surrounded by the skeleton
In the standard case, you write one layout file, but multiple page content files: The idea behind the layouts is the shared design of every page, so you can concentrate on the content instead.

All layout and page content files are Smarty-Templates. Please refer to their documentation for an in-detail description of Smarty's possibilities.

All layout and page content files go to the pages/ directory, or wherever you set the pageDir variable in doc_config.php The filename must be <page-key>.html. The page-key is the key you used in the pages config in doc_config.php.

A very simple page template can look as follows: {literal} 

<h1>Hello, World!</h1>
<p>You are on page '{$page_key}'.</p>
{/literal} which ends in:

Hello, World!

You are on page '{$page_key}'.
