Dynamic page display

Viewing The Page Bakery pages online (dynamically generated) is very simple: Store the whole pagebakery root on your PHP-5.3-enabled web-server and call [pagebakery-root]/index.php. Your page(s) are then displayed in the web broser.

If you want to address some pages directly, use the following request variables

  • index.php?page=[pagename] displays the named page directly.
  • index.php?page=[pagename]&sub=[subpage-name] displays the named page with its subpage directly.
